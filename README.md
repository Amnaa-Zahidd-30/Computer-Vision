VisionLab --- Computer Vision Technology:

A GUI application for image processing built with Python and OpenCV which Implements basic computer vision operations.

Features:
- Grayscale conversion, blur, threshold, edge detection
- Image rotation (45°, 90°) and resizing
- Histogram equalization and sharpening
- Interactive crop tool (click and drag)
- Custom text overlay with fonts and colors
- Image blending
- Pixel statistics analysis using Pandas
- Side-by-side comparison of original and processed images

Technologies:
Python, OpenCV, NumPy, Pandas, Tkinter
